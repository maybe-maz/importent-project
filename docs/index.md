# Smart Attendance System

A web-based attendance management system with a real backend API.

## Features

- Doctor portal: create and manage lectures, set passwords, export attendance
- Student portal: check in to lectures with a lecture password
- REST API backed by a Node.js/Express server

## Getting Started

```bash
npm install
npm start
```

The server runs on `http://localhost:8000` by default.
